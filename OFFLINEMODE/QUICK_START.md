## 🔄 Cara Menggunakan Offline

### Saat Online:
```
1. Buka aplikasi normally
2. Buat transaksi seperti biasa
3. Data otomatis sync ke server
4. Cache terus terupdate
```

### Saat Offline:
```
1. Buka semua menu normally
2. Data dari cache (sangat cepat!)
3. Transaksi baru disimpan lokal
4. Upload otomatis saat online
```

### Transisi Online ↔ Offline:
```
App otomatis mendeteksi:
🟢 Online? → Real-time dari server
🔴 Offline? → Load dari cache lokal
Tidak perlu konfigurasi manual!
```

## 🆘 Troubleshooting

### ❌ "Halaman tidak tersedia"

**Penyebab:**
Halaman belum pernah dibuka saat online

**Solusi:**
```
1. Sambung internet
2. Buka halaman yang error
3. Tunggu loading selesai
4. Offline lagi - halaman sudah cached
```

### ❌ "Data tidak terupdate"

**Penyebab:**
Data lama dari cache belum sinkron

**Solusi:**
```
1. Sambung internet
2. Tunggu beberapa detik (auto-sync)
3. Atau: Dashboard → Pengaturan → Offline Management
   → Klik "Sinkronisasi Sekarang"
4. Data sekarang fresh!
```

### ❌ "Aplikasi crash / error"

**Solusi:**
```
1. F5 atau swipe refresh
2. Tunggu hingga loading selesai
3. Jika masih error, clear cache:
   - Dashboard → Pengaturan → Offline Management
   - Klik "Bersihkan Cache"
   - Reopen di halaman utama
```

## 💾 Manajemen Cache

### Re-Cache Data:
```
Dashboard → Pengaturan → Offline Management
→ Klik "Preload Data Offline"
→ Tunggu sampai selesai
```

### Bersihkan Cache:
```
Dashboard → Pengaturan → Offline Management
→ Klik "Bersihkan Cache"
→ Konfirmasi → Selesai
(Gunakan jika aplikasi lambat atau perlu reset)
```
